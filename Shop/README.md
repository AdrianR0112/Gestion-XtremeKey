# Shop

Frontend ecommerce construido con Next.js App Router, TypeScript y Tailwind CSS.

## Scripts

```bash
npm run dev
npm run build
npm run start
npm run lint
```

## Estructura

- `src/app`: rutas y layouts
- `src/components`: UI y secciones reutilizables
- `src/modules`: acceso a datos, validaciones y stores de dominio
- `src/hooks`: hooks cliente para auth, carrito, productos y pedidos
- `src/lib`: utilidades, constantes y cliente API base
